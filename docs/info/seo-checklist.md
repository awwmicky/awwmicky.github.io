# SEO Checklist

## Ideal Content Info

- title :: max 70 char
- description :: max 200 char

## Ideal Thumbnail Image

- aspect ratio → 16:9 → 1.91:1 → 1200x627
- image type → jpg, png, webp, gif
- image size → max 5mb
- example
  - 200x150, 300x157, 600x315, 640x480, 900x472.5, 1200x630
